# baby-star
Proje Çalışması
PHP tabanlı bebek giyim e-ticaret projesi.

# Proje Özellikleri
- Kullanıcı kayıt / giriş sistemi
- Kız & Erkek bebek ürünleri
- Sepet sistemi (session tabanlı)
- Sahte ödeme ekranı
- Admin panel olmadan ürün listeleme
- Bootstrap ile responsive tasarım

# Kullanılan Teknolojiler
- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

# Proje Yapısı
- `actions/` → Sepet ve ödeme işlemleri  
- `auth/` → Giriş & kayıt  
- `kategori/` → Kız / Erkek ürün sayfaları  
- `css/` → Stil dosyaları  
- `images/` → Görseller
